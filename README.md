# nano-llama

A minimal implementation of the Llama-style Transformer architecture.

## Features (planned)
- RMSNorm
- RoPE (Rotary Positional Embeddings)
- SwiGLU Feedforward
- Grouped Query Attention (GQA)
- Mixture of Experts (MoE)

## Goal
Build a clean, minimal, extensible transformer
